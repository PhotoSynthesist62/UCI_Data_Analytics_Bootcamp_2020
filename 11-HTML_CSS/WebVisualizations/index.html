Latitude Analysis Dashboard

Project Introduction

Hello, and welcome to my Latitude Analysis Dashboard. Although I learned HTML4 back in college, I have not used those skills in many years. This web site is also my first experience with Cascading Style Sheets (CSS). On these pages you will find various scatter plots that were created with Pandas in Jupyter Notebook, using the cities.csv provided. These scatterplots illustrate a number of cities by their latitude against various weather phenomena, including Maximum Temperature, Humidity, Wind Speed and Cloudiness.
