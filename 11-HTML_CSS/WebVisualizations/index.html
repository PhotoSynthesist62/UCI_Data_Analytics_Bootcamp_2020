Introduction

Hello, and welcome to the Latitude Analysis Dashboard created for the HTML/CSS segment of the 2020 UCI Data Analytics Boot Camp. This website is my first experience with Cascading Style Sheets and Bootstrap.

The scatterplot visualizations on these pages were created for an earlier project. ("WeatherPy" in directory 06-Python_APIs) The visualizations illustrate trends among weather phenomena across 500 cities. Phenomena include Maximum Temperature, Humidity, Wind Speed and Cloudiness.
